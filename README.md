# Customer CRM (CSV)
**โครงสร้างโปรเจค**
```
customer_crm/
├── customer_manager      # ฟังก์ชันหลักทั้งหมด
├── main                  # จุดเริ่มโปรแกรม
├── data/
│   └── customers.csv     # ไฟล์ข้อมูลตัวอย่าง (>= 15 แถว)
└── tests/
    ├── test_unit         # Unit tests สำหรับทุกฟังก์ชัน
    └── test_e2e          # E2E test จำลองการใช้งานจริง
```
